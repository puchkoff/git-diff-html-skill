# Changelog

All notable changes to this project are documented here.
Format follows [Keep a Changelog](https://keepachangelog.com/); versioning is [SemVer](https://semver.org/).

## [1.0.0] - 2026-06-17

First tagged release.

### Added
- Render `git diff` as a self-contained HTML page and open it in Safari.
- Inline (unified, GitHub-style) view by default; `--side-by-side` for the two-pane view.
- Light-blue worktree banner and sticky per-file headers.
- Full-file viewer popup with syntax highlighting and in-place change coloring.
- `--no-open` to write the file without opening it.
- `--version` / `-V` flag.
